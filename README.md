Arduino UNO
 	  The Uno is a microcontroller board based on the ATmega328P. It has 14 digital input/output pins (of which 6 can be used as PWM outputs), 6 analog inputs, a 16 MHz quartz crystal, a USB connection, a power jack, an ICSP header and a reset button. It contains everything needed to support the microcontroller; simply connect it to a computer with a USB cable or power it with a AC-to-DC adapter or battery to get started.
	Technical specifications of Arduino UNO :

PROGRAM 
int ledPin = 13; // choose the pin for the LED 
int inPin = 7; // choose the input pin (for a pushbutton) 
int val =0; // variable for reading the pin status
void setup() 
{ 
pinMode(ledPin, OUTPUT); // declare LED outputpinMode
(inPin, INPUT); // declare pushbutton as input
}
void loop()
{ 
val = digitalRead(inPin); // read input value if (val == HIGH) // check if the input is HIGH (button released) 
{
digitalWrite(ledPin, LOW); // turn LED OFF } else 
{ digitalWrite(ledPin, HIGH); // turn LED ON } 
}
